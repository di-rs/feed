# Feed Reader

## ğŸ“ Content Structure

Posts live in `src/content/posts/{YYYY-MM-DD}/{slug}.md` with the following frontmatter:

```yaml
---
title: "Post Title"
date: "2026-02-08"        # ISO format: YYYY-MM-DD
time: "14:30"             # HH:MM format
source: "Blog Name"       # Source publication
link: "https://..."       # Link to original article
image: "https://..."      # Optional image URL
tags: ["tag1", "tag2"]    # Array of tags
---

Your TL;DR content here (2-3 sentences).
```

## ğŸ”§ Customization

### Adding Posts

Posts are automatically generated by a cron job. To manually add a post:

1. Create a new directory: `src/content/posts/YYYY-MM-DD/`
2. Add a markdown file: `your-slug.md`
3. Include required frontmatter (see Content Structure above)
4. Write your TL;DR in the markdown body

### Changing Colors

Edit `src/styles/global.css` to customize the color scheme. The site uses Tailwind's dark mode classes.

### Adjusting Layout

- **Max width:** Change `max-w-6xl` in `src/pages/index.astro`
- **Grid columns:** Modify `grid-cols-1 md:grid-cols-2`
- **Spacing:** Adjust padding/margin utilities
