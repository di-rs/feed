December 2025 was a lighter month for Baseline updates, but still brought several meaningful additions to the web platform. New Baseline Newly available features include the Event Timing API, Largest Contentful Paint API, CSS scrollbar-color, CSS @scope, the scrollend event, and Invoker Commands. Baseline Widely available gains include JavaScript modules in web workers and calc() mathematical keywords.
