January 2026 brought major web platform updates with Chrome 144 and Firefox 147 stable releases. Highlights include CSS Anchor Positioning and the Navigation API becoming Baseline Newly available, the Temporal API landing in Chrome 144 as a modern replacement for the Date object, and a new declarative `<geolocation>` HTML element. View transition enhancements and the `::search-text` pseudo-element for find-in-page styling also shipped.
