A developer reflects on their love for ActionScript 3 and attempts to revive it using Apache Royale, only to find incomplete documentation and a steep MXML learning curve. Despite excitement about Royale's modern features like arrow functions and null conditional operators, the effort to wade through missing docs wasn't justified over modern JavaScript for their use cases.
