SkillShare is a CLI tool that solves the problem of managing AI agent skills across multiple tools (Claude Code, OpenClaw, Codex). By creating a single source directory with symlinks to each AI CLI, it ensures skills stay synchronized across all platforms, with built-in security scanning and team collaboration features.
