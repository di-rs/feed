A2UI (Agent to UI) is a Google protocol that lets AI agents generate interactive user interfaces on demand through declarative mini-apps, similar to Telegram Mini Apps. Instead of endless chat bubbles, agents can send native, structured UI components defined in JSON schemas that work without custom integration code. This article walks through building an A2UI mini-app end-to-end using React and Google's Gemini model.
