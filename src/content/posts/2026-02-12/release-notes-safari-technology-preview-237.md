Safari Technology Preview 237 is now available for macOS Tahoe and macOS Sequoia, bringing a range of fixes and new features. Highlights include support for the `:heading` pseudo-class selector in CSS, multiple accessibility fixes for ARIA attributes and VoiceOver compatibility, and several CSS `@scope` bug fixes. The release covers WebKit changes between commits 305774@main and 306595@main.
