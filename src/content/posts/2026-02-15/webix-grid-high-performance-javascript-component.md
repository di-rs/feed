Webix Grid is an open-source (GPLv3) high-performance JavaScript component designed for displaying and managing large amounts of tabular data in web applications. The component is built for speed with large datasets and offers a feature-rich API for sorting, filtering, and editing grid data. It's released by the Webix team as a standalone open-source offering.
