SVAR React Gantt is a high-performance, MIT-licensed Gantt chart component built with React 19 compatibility, full TypeScript support, and a drag-and-drop interface. It supports task dependencies, hierarchical subtasks, configurable time scales, and handles 10,000+ tasks efficiently. A PRO edition adds auto-scheduling, critical path, undo/redo, and export to PDF/Excel/MS Project.
