A developer documents their first week of the Linux Upskill Challenge, learning remote server administration via WSL connecting to a Google Cloud Platform VM. The log covers SSH key-based authentication, essential health-check commands (uptime, free, df, htop), file permissions, user management, and the basics of systemd service management — all from the perspective of someone moving from "using Linux" to actually managing it.
