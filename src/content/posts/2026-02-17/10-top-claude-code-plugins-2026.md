A curated list of the top 10 MCP plugins for Claude Code, selected from over 1,000 available servers in the ecosystem to supercharge development workflows. Highlights include Awesome Claude Plugins for workflow orchestration, Claude-Mem for persistent cross-session memory, TypeScript & Rust LSP plugins for real-time type checking, and Agent-Peer-Review for having Claude and Codex critique each other's output. These plugins give Claude live access to documentation, databases, and DevOps tooling instead of relying solely on training data.
