Part 2 of this series gets hands-on, building an AI agent from scratch using OpenAI's API in TypeScript — starting with a simple stateless chatbot and progressively adding conversational memory so the LLM retains context across messages. The article highlights the key insight that a basic LLM call starts fresh every time, making true conversation impossible without maintaining message history. It lays the groundwork for more advanced agentic behaviors in future installments.
