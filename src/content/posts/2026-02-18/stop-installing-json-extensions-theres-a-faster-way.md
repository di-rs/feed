Instead of dealing with ad-laden formatters or browser extensions that request broad permissions, the author recommends jsonformat.co — a no-signup tool that formats, validates, and minifies JSON instantly in the browser. It complements CLI tools like jq for situations where you need quick on-screen formatting in meetings, on unfamiliar machines, or when security policy blocks extensions. The article makes the case that a simple bookmark works everywhere, forever.
