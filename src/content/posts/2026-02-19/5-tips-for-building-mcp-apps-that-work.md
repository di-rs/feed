MCP Apps allow AI agents to render interactive UI (charts, forms, video players) directly in the chat interface via iFrames, bridging the gap where clicking a button is clearer than describing an action to an agent. Originally experimental as MCP-UI, the spec is now an official MCP extension supported by Goose, Claude, ChatGPT, and Postman.
