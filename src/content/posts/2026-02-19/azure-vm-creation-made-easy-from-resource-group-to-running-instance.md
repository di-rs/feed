A lean, step-by-step guide to spinning up an Azure Virtual Machine — from subscription setup and resource group creation to a live running instance — covering both Linux (SSH key pair) and Windows (RDP) deployments. The guide emphasizes security-first practices, pay-as-you-go cost efficiency, and includes a tip about Azure for Students which offers free annual credits with no credit card required.
