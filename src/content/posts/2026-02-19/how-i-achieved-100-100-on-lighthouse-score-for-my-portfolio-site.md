A developer shares how they achieved a perfect 100/100 Lighthouse score on their Next.js portfolio by making targeted performance optimizations. Key changes include replacing scroll event listeners with the Intersection Observer API to reduce main thread blocking, and fixing LCP animation traps that delay the largest contentful paint. The article provides practical, copy-paste-ready techniques any developer can apply to significantly boost their own site's performance metrics.
